# Meu Primeiro PR

Este repositório foi criado para praticar o fluxo de **Pull Request** no GitHub.

## Sobre

Aqui eu aprendi, na prática, a:

1. Criar uma branch
2. Fazer um commit com minhas alterações
3. Enviar (push) para o GitHub
4. Abrir um Pull Request
5. Revisar e fazer o merge
